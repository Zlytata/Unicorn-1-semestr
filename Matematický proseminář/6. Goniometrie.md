## I. Oblouková Míra, Stupně a Radiány

Pro popis velikosti úhlu se primárně používají dvě jednotky: **stupně** a **radiány** (oblouková míra).

* **Důležitá Definice: Radián**
    V obloukové míře **velikost 1 radiánu (1 rad)** vymezuje na jednotkové kružnici (kružnici o poloměru 1) oblouk délky **1**.
    * Obecně pro kružnici o poloměru $r$ odpovídá 1 rad oblouku délky $r$.
* **Převody úhlů:**
    * **Plný úhel** je $360^\circ$, což odpovídá $2\pi$ radiánům.
    * **Přímý úhel** je $180^\circ$, což je zároveň **$\pi$ radiánů**.
    * **Pravý úhel** je $90^\circ$, což je $\frac{\pi}{2}$ radiánů.
* **Kladná orientace úhlu** je **proti směru hodinových ručiček**.

| Stupně | Radiány ($\pi$) |
| :---: | :---: |
| $0^\circ$ | $0$ |
| $30^\circ$ | $\pi/6$ |
| $45^\circ$ | $\pi/4$ |
| $60^\circ$ | $\pi/3$ |
| $90^\circ$ | $\pi/2$ |
| $180^\circ$ | $\pi$ |
| $270^\circ$ | $3\pi/2$ |
| $360^\circ$ | $2\pi$ |

## II. Goniometrické Funkce (Sinus, Kosinus, Tangens, Kotangens)

### 1. Sinus (SIN)
* **Značení:** $f: y = \sin(x)$
* **Definiční obor:** $D_f = \mathbb{R}$
* **Obor hodnot:** $H_f = \langle -1; 1 \rangle$
* **Perioda:** $2\pi$
* **Parita:** **Lichá** funkce ($\sin(-x) = -\sin(x)$)
* **Spojitost:** Spojitá na $D_f$.
* **Nulové body:** $\sin(k\pi) = 0, k \in \mathbb{Z}$
* **Lokální extrémy (Max/Min):** Maxima v $\frac{\pi}{2} + 2k\pi$, Minima v $\frac{3\pi}{2} + 2k\pi$.

### 2. Kosinus (COS)
* **Značení:** $f: y = \cos(x)$
* **Definiční obor:** $D_f = \mathbb{R}$
* **Obor hodnot:** $H_f = \langle -1; 1 \rangle$
* **Perioda:** $2\pi$
* **Parita:** **Sudá** funkce ($\cos(-x) = \cos(x)$)
* **Spojitost:** Spojitá na $D_f$.
* **Nulové body:** $\cos(\frac{\pi}{2} + k\pi) = 0, k \in \mathbb{Z}$
* **Lokální extrémy (Max/Min):** Maxima v $2k\pi$, Minima v $\pi + 2k\pi$.

### 3. Tangens (TAN)
* **Značení:** $f: y = \tan(x)$
* **Definiční obor:** $D_f = \mathbb{R} \setminus \{\frac{\pi}{2} + k\pi; k \in \mathbb{Z}\}$
* **Obor hodnot:** $H_f = \mathbb{R}$
* **Perioda:** $\pi$
* **Parita:** **Lichá** funkce ($\tan(-x) = -\tan(x)$)
* **Asymptoty:** Ve vyloučených bodech $x = \frac{\pi}{2} + k\pi$.
* **Monotonie:** **Rostoucí** v intervalech $(-\frac{\pi}{2} + k\pi; \frac{\pi}{2} + k\pi)$.
* **Lokální extrémy:** Nemá lokální maxima ani minima.
* **Nulové body:** $\tan(k\pi) = 0, k \in \mathbb{Z}$

### 4. Kotangens (COT)
* **Značení:** $f: y = \cot(x)$
* **Definiční obor:** $D_f = \mathbb{R} \setminus \{k\pi; k \in \mathbb{Z}\}$
* **Obor hodnot:** $H_f = \mathbb{R}$
* **Perioda:** $\pi$
* **Parita:** **Lichá** funkce ($\cot(-x) = -\cot(x)$)
* **Asymptoty:** Ve vyloučených bodech $x = k\pi$.
* **Monotonie:** **Klesající** v intervalech $(k\pi; \pi + k\pi)$.
* **Lokální extrémy:** Nemá lokální maxima ani minima.
* **Nulové body:** $\cot(\frac{\pi}{2} + k\pi) = 0, k \in \mathbb{Z}$

### Hodnoty základních funkcí

| $x$ | $0$ | $\pi/6$ ($30^\circ$) | $\pi/4$ ($45^\circ$) | $\pi/3$ ($60^\circ$) | $\pi/2$ ($90^\circ$) |
| :---: | :---: | :---: | :---: | :---: | :---: |
| $\sin x$ | $0$ | $\frac{1}{2}$ | $\frac{\sqrt{2}}{2}$ | $\frac{\sqrt{3}}{2}$ | $1$ |
| $\cos x$ | $1$ | $\frac{\sqrt{3}}{2}$ | $\frac{\sqrt{2}}{2}$ | $\frac{1}{2}$ | $0$ |
| $\tan x$ | $0$ | $\frac{\sqrt{3}}{3}$ | $1$ | $\sqrt{3}$ | Nedefinováno |
| $\cot x$ | Nedefinováno | $\sqrt{3}$ | $1$ | $\frac{\sqrt{3}}{3}$ | $0$ |

* **Jednotková kružnice:** Pro určení hodnot včetně znamének se používá **jednotková kružnice**. $\sin \alpha$ odpovídá **$y$-souřadnici** bodu, jehož průvodič svírá s osou $x$ úhel $\alpha$, a $\cos \alpha$ odpovídá **$x$-souřadnici**.

## III. Goniometrické Vzorce

### Vzájemné vztahy

* $\sin x = \cos \left( x - \frac{\pi}{2} \right)$
* $\cos x = \sin \left( x + \frac{\pi}{2} \right)$
* $\tan x = \frac{\sin x}{\cos x}$
* $\cot x = \frac{\cos x}{\sin x} = \frac{1}{\tan x}$

### Součet druhých mocnin (Pythagorova věta pro goniometrické funkce)
* $$\sin^2 x + \cos^2 x = 1$$
    *(Pozor: $\sin^2 x = (\sin x)^2$, což $\ne \sin(x^2)$)*

### Periodicita
* $\sin x = \sin(x \pm 2k\pi)$
* $\cos x = \cos(x \pm 2k\pi)$
* $\tan x = \tan(x \pm k\pi)$
* $\cot x = \cot(x \pm k\pi)$

### Součtové vzorce
* $\sin(x \pm y) = \sin x \cos y \pm \cos x \sin y$
* $\cos(x \pm y) = \cos x \cos y \mp \sin x \sin y$

### Dvojnásobný úhel
* $\sin(2x) = 2 \sin x \cos x$
* $\cos(2x) = \cos^2 x - \sin^2 x$

## IV. Goniometrické Rovnice

Goniometrické rovnice jsou takové rovnice, kde se **neznámá vyskytuje v argumentu** některé goniometrické funkce.

### Obecná strategie řešení

1.  **Zjednodušení výrazů** v rovnici s cílem získat **tvar 'goniometrická funkce(výraz s $x$) = číselný výraz'**.
2.  **Vyjádření nutných podmínek** rovnosti (pomocí periodicity).
3.  Závěrečné úpravy (pro izolaci $x$).

Při úpravě goniometrických rovnic používáme všechny běžné úpravy rovnic a goniometrické vzorce. Kořeny se v důsledku periodicity vždy vyjadřují jako množina.
