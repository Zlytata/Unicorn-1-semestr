**Object** je jediný **komplexní (složený) datový typ** (_complex data type_) v JavaScriptu. Slouží k ukládání kolekcí dat a složitějších entit. Zatímco primitivní typy ukládají jednu jedinou hodnotu (např. řetězec nebo číslo), objekt může ukládat mnoho hodnot a složité logiky.

### 1. Struktura Objektu: Klíče a Vlastnosti

Objekt je v podstatě **kolekce párů klíč-hodnota** (_key-value pairs_).

#### a) Vlastnost (_Property_)

Celý pár, skládající se z klíče a asociované hodnoty, se nazývá **vlastnost** (_property_).

#### b) Klíč (_Key_) / Název Vlastnosti (_Property Name_)

- **Klíč** je jméno, které jednoznačně identifikuje data v objektu.
    
- Klíče jsou v moderním JavaScriptu vždy **řetězce** (_String_) nebo (méně často) typ **Symbol**.
    

#### c) Hodnota (_Value_)

- **Hodnota** je data uložená pod daným klíčem.
    
- Hodnotou může být **cokoliv**: primitivní typ (_String_, _Number_, _Boolean_, atd.), **funkce** (pak mluvíme o **metodě**) nebo dokonce **jiný objekt**.
    
```javascript
let uzivatel = {
  // KLÍČ: "jmeno", HODNOTA: "Jan" (String)
  jmeno: "Jan", 
  
  // KLÍČ: "vek", HODNOTA: 30 (Number)
  vek: 30,
  
  // VLASTNOST "pozdrav" je METODA (funkce)
  pozdrav: function() {
    return "Ahoj, jsem " + this.jmeno;
  }
};
```

### 2. Vytvoření Objektu

Nejběžnější způsob vytvoření objektu je pomocí **literálu objektu** (_object literal_):

```javascript
const osoba = {
  jméno: "Petra",
  příjmení: "Nová",
  adresa: {
    město: "Praha"
  }
};
```

### 3. Všechno je Objekt (Téměř)

V JavaScriptu většina věcí, které nejsou primitivní typy, dědí z _Object_. Do kategorie objektů tedy spadají i:

- **Pole (_Arrays_):** Speciální objekty s číselnými klíči pro indexy.
    
- **Funkce (_Functions_):** Mají vlastnosti a metody, i když se jejich `typeof` vrací `"function"`.
    
- **Regulární výrazy (_RegExp_)**
    
- **Datum (_Date_)**
    

### 4. Dynamická Povaha

Objekty jsou **proměnné** (_mutable_). Po jejich vytvoření je možné **přidávat, odebírat nebo měnit** jejich vlastnosti, a to i v případě, že jsou deklarovány pomocí `const`.

```javascript
const auto = { barva: "červená" };

// Změna existující vlastnosti:
auto.barva = "modrá"; 

// Přidání nové vlastnosti:
auto.rok = 2024;
```

_(Poznámka: Proměnnou `auto` samotnou nelze znovu přiřadit, ale její obsah (vlastnosti) se měnit smí.)_

### 5. Přístup k Vlastnostem

K hodnotám vlastností se přistupuje dvěma způsoby:

|**Syntax**|**Popis**|**Příklad**|
|---|---|---|
|**Tečková notace** (`.`)|Používá se, když **znáte** název vlastnosti (klíč).|`uzivatel.jmeno` // Vrací "Jan"|
|**Hranatá závorka** (`[]`)|Používá se, když je název vlastnosti uložen v **proměnné** nebo obsahuje mezery/speciální znaky.|`uzivatel["vek"]` // Vrací 30|
