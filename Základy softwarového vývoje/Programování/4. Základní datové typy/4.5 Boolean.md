
---

Datový typ **Logická hodnota** (_Boolean_) je primitivní datový typ, který reprezentuje jedinou logickou hodnotu: buď **pravda** (_true_) nebo **nepravda** (_false_). Je to základní stavební kámen pro rozhodování a řízení toku programu (_control flow_).

### 1. Dvě Hodnoty

Typ _Boolean_ má pouze dvě možná klíčová slova/hodnoty:

1. **`true`** (pravda)
    
2. **`false`** (nepravda)
    

### 2. Použití

Logické hodnoty jsou klíčové pro:

- **Podmíněné příkazy** (_Conditional Statements_): Používají se v konstrukcích jako `if`, `else`, a `switch`.
    
- **Smyčky** (_Loops_): Určují, zda má smyčka (_for_, _while_) pokračovat v opakování kódu.
    
- **Logické operátory** (_Logical Operators_): Například `&&` (a), `||` (nebo) a `!` (negace).
    

```javascript
let jePlatny = true;

if (jePlatny) {
  // Tento kód se provede, protože jePlatny je true
}
```

### 3. Logické Přetypování (_Boolean Coercion_)

V JavaScriptu jsou hodnoty často **implicitně** převedeny na logický typ (_Boolean_) v kontextech, kde se očekává logická hodnota (např. v `if` podmínce).

Všechny hodnoty v JavaScriptu jsou rozděleny do dvou skupin:

#### a) Falešné Hodnoty (_Falsy Values_)

Jedná se o 6 hodnot, které se při převodu na _Boolean_ vyhodnotí jako **`false`**:

|**Hodnota**|**Typ**|**Popis**|
|---|---|---|
|**`false`**|_Boolean_|Samotná hodnota nepravdy.|
|**`0`** (nula)|_Number_|Číselná nula (včetně `-0`).|
|**`""`**|_String_|Prázdný řetězec.|
|**`null`**|_Null_|Speciální hodnota pro "nic".|
|**`undefined`**|_Undefined_|Nedefinovaná hodnota.|
|**`NaN`**|_Number_|_Not-a-Number_ (Není číslo).|

#### b) Pravdivé Hodnoty (_Truthy Values_)

**Všechny ostatní hodnoty** se při převodu na _Boolean_ vyhodnotí jako **`true`**.

- **Příklady:**
    
    - Jakékoliv kladné nebo záporné číslo (kromě 0), např. `1`, `-10`.
        
    - Jakýkoliv řetězec, který není prázdný, např. `"false"`, `"0"`.
        
    - Všechny **objekty** (_Object_), **pole** (_Array_) a **funkce** (_Function_).
        

```javascript
// Příklad Falsy hodnoty:
if (0) {
  // Kód se neprovede, protože 0 je Falsy.
}

// Příklad Truthy hodnoty:
if ("Ahoj") {
  // Kód se provede, protože neprázdný řetězec je Truthy.
}
```

### 4. Funkce `Boolean()` a Operátor Negace

Pro explicitní převod na _Boolean_ lze použít funkci **`Boolean(hodnota)`** nebo operátor **dvojité negace** (**`!!`**):

```javascript
let cislo = 10;

// Explicitní převod funkcí:
let jePravda_A = Boolean(cislo); // true

// Explicitní převod dvojitou negací (častá zkratka):
let jePravda_B = !!cislo; // true
```
