## Přiřazení hodnoty (_Variable Inicialization_)

**Přiřazení do proměnné** (_variable inicialization) slouží k nastavení určité **hodnoty** (_value_) dané proměnné. S přiřazením se setkáte v podstatě v rámci různých algoritmů.

Nejjednodušší zápis pro přiřazení je pomocí rovnítka:

```javascript
x = 3;
```

Tento zápis říká, že se má do proměnné **x přiřadit hodnota 3**.

Často se setkáte se spojením **deklarace** (_declaration_) a **přiřazení** (_assignment_) zároveň:

```javascript
let y = 3;
```

### Znak rovnítka (`=`)

> ℹ️ Zatímco v matematice znak **`=`** vyjadřuje **rovnost** (_equality_) dvou výrazů (levé a pravé strany), v programovacích jazycích tento znak znamená pokyn k **„přiřazení pravé strany do levé“** (_assignment_). Po (úspěšném) provedení takovéhoto řádku sice skutečně platí ona matematická rovnost mezi levou a pravou stranou, ale to je až přirozený důsledek přiřazení. Rovnítko **není nicméně skutečný příkaz** (_statement_), nikoliv vyjádřením stavu věci.

## Použití proměnných (_Variable Usage_)

Pro **výpis jakékoliv informace** (_outputting information_) v rámci ukázek se bude používat příkaz **`console.log()`**.

Základní informace je, že cokoliv se vloží do **`console.log()`**, se **vypíše** (_is printed_). V příkladu výše se vypíše cokoliv, co je aktuálně nastaveno v proměnné **`x`**.

- Pokud chcete vypsat například text "Hello World", můžete zavolat příkaz: `console.log("Hello World")`.
    
- Pokud chcete vypsat hodnotu proměnné `x`, použijete: `console.log(x)`.
    

### Změna hodnoty a řetězce s proměnnými

Hodnotu proměnné lze kdykoliv přepsat:

```javascript
let x = 3; 
// x je 3
x = 5; 
// x je nyní 5
```

Hodnoty proměnných si lze **vzájemně přiřazovat**:

```javascript
let y = 4;
let x = y; // x dostane hodnotu y (což je 4)
```

Pro **konzoli** (_console_) můžeme **vypisovat proměnnou a text** (_output variable and text_) jiným způsobem (tzv. **template literal** s dolarem a složenými závorkami – `${x}`). To umožňuje vkládat **atributy** (_attributes_) a **hodnoty** (_values_) přímo do obyčejného textu, který je pak vypisován jako **řetězec** (_string_).

```javascript
let x = 3;        // assign 3 to x
console.log(x);   // output the value of x
let y = 4;        // assign 4 to y
console.log(y);   // output the value of y
x = y;            // assign the value of y (which is 4) to x
console.log(x);   // output the value of x
console.log(aaa); // error
```

```javascript
//Cvičení
const x = 4, const y = 5 //const protože nehodlám měnit hodnotu
const sum = x + y;
console.log(sum); 
```
