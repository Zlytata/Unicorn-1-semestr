- **Výraz** (_Expression_) je v programování jakákoliv platná jednotka kódu, která se po vyhodnocení převede na jednu jedinou **hodnotu**.
    
- Může jít o jednoduchou konstantu nebo složitou kombinaci proměnných, operátorů a volání funkcí.
    

**Příklady:**

|**Typ Výrazu**|**Příklad**|**Hodnota (Výsledek Vyhodnocení)**|
|---|---|---|
|**Konstanta / Literál**|`10`, `"ahoj"`|Číslo 10, Text "ahoj"|
|**Proměnná**|`hrubaMzda`|Hodnota uložená v proměnné|
|**Aritmetické operace**|`x + y * 2`|Výsledná číselná hodnota|
|**Logické operace**|`true` nebo `x > 5`|Pravdivostní hodnota (`true` nebo `false`)|

> ⚠️ **Doporučení:** Vyhýbejte se psaní příliš složitých výrazů, které kombinují výpočty s přiřazením (_assignment_). Takový kód je náročný na čtení a náchylný k chybám.

---

### 2. Příkaz Přiřazení (_Assignment Statement_)

**Definice:**

- **Příkaz přiřazení** (_Assignment Statement_) slouží k uložení hodnoty do proměnné pomocí **operátoru přiřazení** (`=`).
    

**Přiřazení jako Výraz:**

- Samotné **přiřazení** (např. `a = 5`) je považováno **také za výraz** (_expression_).
    
- **Hodnota** celého výrazu přiřazení je hodnota, která byla právě přiřazena.
    
    - _Příklad:_ Výraz `a = 5` se vyhodnotí na hodnotu **5**.
        
- Tato vlastnost umožňuje **řetězové přiřazení** (_Chained Assignment_), kdy se jedna hodnota přiřadí více proměnným najednou:
    
    - _Příklad:_ `a = b = 5`. Hodnota 5 je nejprve přiřazena `b`, a hodnota tohoto výrazu (5) je pak přiřazena `a`.

### 3. Příklad – Výpočet Průměru

**Úkol:** Vypočítat průměr ze tří hodnot ($\mathbf{a}$, $\mathbf{b}$, $\mathbf{c}$) a výsledek uložit do proměnné $\mathbf{avg}$.

**JavaScript Řešení:**

```javascript
const a = 8;
const b = 12;
const c = -2;

// (a + b + c) / 3 je Expression (výraz)
const avg = (a + b + c) / 3; // Celý řádek je Assignment Statement (příkaz přiřazení)

console.log(avg); // Výstup: 6
```
