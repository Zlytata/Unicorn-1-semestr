Komentáře slouží v kódu k zapsání poznámek, vysvětlení, jak kód funguje, nebo k zaznamenání úkolů, které je potřeba dokončit. Pro _interpreter_ je komentovaná část kódu neviditelná a je při zpracování přeskočena.

### 1. Typy Komentářů

V JavaScriptu existují dva základní typy syntaxe pro komentování:

|**Název**|**Značení**|**Použití**|**Příklad**|
|---|---|---|---|
|**Jednořádkový** (_Single-line_)|Dvě lomítka (`//`)|Komentuje text od značky až do konce řádku. Používá se pro krátké poznámky.|`let a = 12; // nastavení proměnné`|
|**Víceřádkový** (_Multi-line_)|Lomítko a hvězdička (`/* */`)|Komentuje blok textu, který může sahat přes více řádků. Je nutné blok **zahájit** (`/*`) a **ukončit** (`*/`).|`/* Tento blok kódu \n se neprovede */`|

### 2. Speciální Komentáře pro Úkoly (_TODO / FIXME_)

Jedná se o konvenci používanou v mnoha technologiích, kde komentáře začínající klíčovými slovy **TODO** nebo **FIXME** označují specifické stavy kódu:

- **`// TODO:`**
    
    - Označuje kód, který je nekompletní nebo vyžaduje dodělání (např. přesun kódu jinam, doplnění funkcionality).
        
- **`// FIXME:`**
    
    - Označuje kód, který momentálně nefunguje správně nebo obsahuje chybu, kterou je potřeba opravit.
        

> Tyto speciální komentáře často dokážou vizualizovat a vyhledat vývojová prostředí (_IDE_) nebo nástroje pro správu kódu.

### 3. Dokumentační Komentáře (_JSDoc_)

**JSDoc** je standard pro dokumentování kódu v JavaScriptu. Používá se pro popis funkcí, tříd a metod, generování externí dokumentace a pro lepší _type-checking_ v moderních editorech kódu.

JSDoc komentáře používají syntaxi víceřádkového komentáře, ale začínají **dvěma hvězdičkami** (`/**`), čímž se odlišují od běžných víceřádkových komentářů.

```javascript
/**
 * Vypočítá součet dvou čísel.
 * * @param {number} num1 První sčítanec.
 * @param {number} num2 Druhý sčítanec.
 * @returns {number} Součet obou čísel.
 */
function soucet(num1, num2) {
  return num1 + num2;
}
```

**Klíčové JSDoc značky (_Tags_):**

- `@param {typ} jméno` – Popisuje vstupní argument (parametr) funkce.
    
- `@returns {typ}` – Popisuje typ vrácené hodnoty funkce.
    
- `@type {typ}` – Popisuje typ proměnné.
    
- `@author` – Uvádí autora kódu.
    

### 4. Příklady Použití

**Příklad Prohození Hodnot** (Z cvičení)

Při prohazování hodnot proměnných **a** a **b** je typicky nutná **pomocná proměnná** (_temporary variable_), což je dobré v kódu okomentovat.

```javascript
let a = 8;
let b = 12;

// Prohození hodnot
let temp = a;  // 1. Uložení hodnoty 'a' do dočasné proměnné
a = b;         // 2. Nastavení 'a' na hodnotu 'b' (12)
b = temp;      // 3. Nastavení 'b' na původní hodnotu 'a' (8)

console.log(a); // Měl by vypsat hodnotu 12
console.log(b); // Měl by vypsat hodnotu 8
```

```javascript
//Cvičení
const a = 4;
const b = 10;
const n = 12;

let perimeter = 2*a + 2*b; //let je použito protože nadále hodlám měnit hodnotu

perimeter = perimeter * n;
```
