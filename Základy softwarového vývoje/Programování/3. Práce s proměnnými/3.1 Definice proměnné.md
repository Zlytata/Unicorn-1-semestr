## Co je proměnná?

**Proměnná** (_Variable_) je obtížně definovatelný pojem. Lze ji popsat jako jakousi **"entitu"** (_entity_), která obsahuje určitou **hodnotu** (_value_). Z nízkoúrovňového pohledu by se dalo říct, že proměnná je pouze označením pojmenovávajícím určité části **paměti počítače** (_computer memory_).

---

## Deklarace proměnné (_Variable Declaration_)

Pokud chceme s proměnnou pracovat, musíme o ní nejprve dát **programovacímu jazyku** (_programming language_) – v tomto případě **JavaScriptu** – vědět. Tato technika se nazývá **deklarace proměnné** (_variable declaration_).

Technicky to znamená, že si programovací jazyk musí **rezervovat daný prostor** pro definovanou hodnotu a držet si **odkaz** (_reference_) na tento prostor pro další práci.

### Praktické způsoby deklarace v JavaScriptu

V JavaScriptu lze proměnnou deklarovat několika způsoby:

1. **`let`**
    
    - Moderní způsob deklarace proměnné.
        
2. **`const`** (Konstanta)
    
    - Deklarace, která říká, že bude možné proměnnou číst, ale **není možné ji měnit** (_immutable_). Tato deklarace by se měla používat společně s přiřazením (_assignment_) hodnoty.
        
3. **`var`**
    
    - **Starší způsob zápisu deklarace** v JavaScriptu.
        

```javascript
//Cvičení
let name = "Jan";
const surname = "Novák";
name = `Jan ${surname}`; //popřípadě natvrdo Novák
```

## Dovysvětlení a tipy navíc

### `let` vs. `var` – proč na `var` zapomenout

- `var` má tzv. *function scope* (platí jen uvnitř funkce), zatímco `let` a `const` mají *block scope* (platí jen uvnitř složených závorek `{}`).
- `var` se „hoistuje“ – proměnná je dostupná před deklarací, ale s hodnotou `undefined`, což může vést k chybám.
- `let` a `const` se také hoistují, ale do tzv. *temporal dead zone*, tudíž k nim nemůžeme přistupovat před jejich deklarací.
- Moderní kód používá `let`/`const` pro předvídatelné chování.

### `const` neznamená neměnnost objektu

- `const` brání jen přepsání reference, ne změně obsahu objektu/array:

```javascript
const user = { name: "John" };
user.name = "Pete"; // OK
user = { name: "Mary" }; // Chyba

        ```
