
* **Multiplatformní** jazyk  
* **Objektově orientovaný** (s prototypovou dědičností)  
* **Událostmi řízený** skriptovací jazyk  
* Autorem je **Brendan Eich** (1995)  
* Inspirován jazyky **C / C++ / Java**

---

## Historie

* Vznikl v roce **1995** jako doplněk k jazykům HTML a Java pro oživení webových stránek  
* **1997** – standardizován asociací **ECMA** (*European Computer Manufacturers Association*)  
* **1998** – převzat do standardů **ISO** (*International Organization for Standardization*)  
* Oficiální název: **ECMAScript**  
* Nejnovější verze: **ECMAScript 2025 (ES16)**

---

## JavaScript

* Vytvořen za účelem **interaktivity webových stránek**  
* Programy v tomto jazyce se nazývají **skripty**  
* Lze jej vložit přímo do **HTML**:

```html
<script>
  const x = 5;
  console.log(x);
</script>
```
* Používá se jak na **klientovi** (prohlížeč), tak na **serveru** (např. Node.js)
* Spustitelný všude, kde je nainstalovaný **JavaScript Engine**

    ### Co JavaScript umožňuje ###
    * Měnit obsah stránky a upravovat styly (DOM manipulace)
    * Reagovat na akce uživatele (kliknutí, stisk klávesy, pohyb myši)
    * Pracovat s **místním úložištěm** (`localStorage`, `sessionStorage`)
    * Odesílat a přijímat data ze serveru (AJAX, Fetch API, WebSockets)
    * Získávat a nastavovat **cookies**

## JavaScript engine

* **JavaScript engine** je program (součást prohlížeče nebo runtime prostředí), který provádí kód napsaný v JavaScriptu.

    ### Hlavní úkoly engine
    * **Parsování (parsing):** převede zdrojový kód na abstraktní syntaktický strom (AST)
    * **Interpretace / kompilace:** přeloží kód do strojových instrukcí. Moderní enginy používají **JIT kompilaci** (*Just-In-Time*)
    * **Správa paměti:** automatické uvolňování nepoužívaných objektů (*garbage collection*)
    * **Optimalizace:** dynamické zrychlování často používaného kódu
    
    ### Příklady známých JavaScript enginů
    * **V8** – Google, Opera (Chrome, Node.js)
    * **SpiderMonkey** – Mozilla (Firefox)
    * **Chakra** – Microsoft Edge
    * **JavaScriptCore (Nitro)** – Safari (Apple)
