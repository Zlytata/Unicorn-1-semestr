### Definice Sekvence

**Sekvence** je základní komponenta JSP a **obsahuje dvě či více částí vyskytujících se v přesném pořadí za sebou**.

Prostřednictvím sekvence ve struktuře programu nebo dat vyjadřujeme:

1.  **Program:** Posloupnost instrukcí programu v přesném pořadí.
2.  **Data:** Posloupnost datových položek v přesném pořadí.

### Příklad datové struktury: Faktura

Datový záznam **faktury** je představován sekvencí **5** částí:

1.  Elementární položka **číslo faktury** (`number`).
2.  Elementární položka **datum vystavení faktury** (`issueDate`).
3.  Elementární položka **datum splatnosti** (`paymentTo`).
4.  Složená položka **dodavatel** (`supplier`).
5.  Složená položka **odběratel** (`customer`).

**Složené položky** jsou dále dekomponovány:

* **Složená položka dodavatel** (`supplier`) je sekvence dvou částí:
    * Elementární položka **název** (`nameSupplier`).
    * Elementární položka **DIČ** (`taxNumber`).
* **Složená položka odběratel** (`customer`) je rovněž sekvence dvou částí:
    * Elementární položka **název** (`nameSupplier`).
    * Elementární položka **DIČ** (`taxNumber`).

**Příklad JSON struktury faktury:**

```json
{
  "number": "FA201900123",
  "issueDate": "12.07.2019",
  "paymentTo": "19.07.2019",
  "supplier": {
    "name": "Unicorn College s.r.o.",
    "taxNumber": "CZ699004029"
  },
  "customer": {
    "name": "Alza.cz a.s.",
    "taxNumber": "CZ27082440"
  }
}
```
