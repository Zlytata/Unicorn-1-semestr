### Definice Selekce

**Selekce** je základní komponenta JSP a **představuje výběr z možností, kdy v jeden okamžik nastane pouze jedna z nich**.

Selekce se používá pro podmíněné zpracování instrukcí nebo výběr z datových struktur.
![[1.4 Obecný formát selekce.png]]

***

### Realizace v JavaScriptu

V JavaScriptu je selekce reprezentována příkazy:

* **`if` / `else if` / `else`**: Pro testování podmínek.
* **`switch`**: Pro výběr jedné z více možností na základě hodnoty proměnné.

#### Příklad datové struktury transakce a implementace

Předpokládáme datovou strukturu finanční transakce (`transaction`) s `transactionCode` pro identifikaci typu:
* **C** (`credit`) - příjem
* **D** (`debit`) - výdej
* **T** (`transfer`) - převod

**Implementace (za předpokladu, že data jsou vždy validní):**

```javascript
if (transactionCode === "C") {
    processCredit(transactionData);
} else if (transactionCode === "D") {
    processDebit(transactionData);
} else if (transactionCode === "T") {
    processTransfer(transactionData);
}```

```javascript
switch (transactionCode) {
    case "C":
        processCredit(transactionData);
        break;
    case "D":
        processDebit(transactionData);
        break;
    case "T":
        processTransfer(transactionData);
        break;
}``` 

![[1.4 Selekce bez erroru.png]]

**Správné řešení (zahrnující ošetření chyb/nevalidních dat):**

Aby bylo řešení správné (ošetření chybných hodnot), je potřeba přidat čtvrtou možnost (`error`).

```javascript
if (transactionCode === "C") {
    processCredit(transactionData);
} else if (transactionCode === "D") {
    processDebit(transactionData);
} else if (transactionCode === "T") {
    processTransfer(transactionData);
} else {
    processError(transactionData);
}```

```javascript
switch (transactionCode) {
    case "C":
        processCredit(transactionData);
        break;
    case "D":
        processDebit(transactionData);
        break;
    case "T":
        processTransfer(transactionData);
        break;
    default:
        processError(transactionData);
```
